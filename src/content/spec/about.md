## 关于网站

本网站使用 [Astro](https://astro.build/) 构建，使用 [Tailwind CSS](https://tailwindcss.com/) 作为 CSS 框架，使用 [Framer Motion](https://www.framer.com/motion/) 作为动画库，使用 [Jotai](https://jotai.org/) 作为状态管理库，是对 [lxchapu](https://github.com/lxchapu) 的 Astro 模板 [astro-gyoza](https://github.com/lxchapu/astro-gyoza) 的二次开发，构建在 Github Actions 上，使用 Github Pages 进行部署。
